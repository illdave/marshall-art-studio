The Options Toolbox® ("Toolbox") is a stand-alone, self contained educational software product. Upon installation, all files that Toolbox writes locally to a hard drive are placed solely in a directory which the product creates for itself. All of these installed files are used only by Toolbox, and not by a computer's operating system. These files are not shared files, and therefore should not interfere with any other installed application. Toolbox does not integrate with any databases or other files beyond those it has installed in its own directory.

Year 2000 Readiness Disclosure

Y2K Compliance for the Options Toolbox®
Toolbox is programmed to offer the end user a choice of future option expirations for analytical purposes, and the number of days from the present date until the selected expiration. The program determines the date of its use and offers future expiration months (and years) forward from that date by reading the operating system's clock. Since Summer of 1997, Toolbox has recognized the existence of year 2000, and has been analyzing options that expire beyond January 1, 2000. In mid-Summer 1998, Toolbox recognized and began analyzing options expiring in year 2001.

The CBOE has tested Toolbox to the best of our ability, and believe all versions (1.1 and above, including Index Toolbox) to be Y2K compliant. CBOE has repeatedly tested the efficacy of Toolbox with respect to its ability to recognize and analyze options expiring well into the 21st century.

The vendors who created the authoring application used to build Toolbox and the consultants who provided the calculation engine have certified their products to be Y2K compliant as well. The statements from these vendors are on file with CBOE.
