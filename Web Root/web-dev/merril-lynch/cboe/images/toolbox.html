Introduction

Whether you use this online version of The Options Toolbox™ or download the Windows version, this educational program is designed to enhance your understanding of exchange-traded equity options, index options and LEAPS® and the many strategies that can be used for fine tuning your investments. However, before starting the online version of the program, there are a few important things to consider:

Do you have the Authorware Web Player 6.0 - Full Version plug-in installed?
The online version of this program requires a recent version of Macromedia's Authorware Web Player Plug-in.

If you are running Windows 95, 98, 2000 or NT, and you do not see the rising sun in the box below, click here to download and install this plug-in.

If you are running a Macintosh OS and do not see the rising sun in the box below, click here to visit the Macromedia web site to download and install the Authorware Web Player 6.0 - Full Version plug-in.

If you don't see the rising sun above you will have to download and install the Macromedia Web Player Software.

  • Windows users click here to download and install the software now.
  • Macintosh users click here to visit the Macromedia Web Site in order to download and install the software.

To use the Online Toolbox you need to allow trusting mode.
Before you can use the Toolbox a dialog box will appear prompting you to choose whether or not to "trust" the application. You must accept trusting mode in order to run the program.The Toolbox uses your hard drive for temporary files only and will not write anything permanently to the disk.
