Cambridge Comix Festival

Web Design and Development, 2003 Website for multi-venue New England comic book convention organized by Hans Rickheit. Event featured 30+ local comic creators exhibiting work, networking with fans and press, and selling their publications.

Transformed organizer's HTML wireframe into polished website balancing public accessibility with the event's creative energy. Created custom illustrations and web graphics, coordinated with participating artists to collect portfolio thumbnails for member section.

Note: This site reflects early 2000s web design practices, built during the table layout era before CSS-based responsive design became standard.
