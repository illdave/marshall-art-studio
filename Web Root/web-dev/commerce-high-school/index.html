Commerce High School

Personal Project, 2000 Website for my 20th high school reunion. Collaborated with alumni director on content, then designed and coded the site, wrote copy, and created custom illustrations including an updated version of my 1980 yearbook cover art.

Features included illustrated quote columns and a hand-coded HTML crossword puzzle customized for my graduating class—"an updated version of our yearbook Activities page."

Note: This site reflects early 2000s web design practices, built during the table layout era before CSS-based responsive design became standard.
