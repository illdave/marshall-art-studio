WBZ Head for Home

Web Design and Development, 2006 Complete architectural rebuild of marketing website for WBZ radio station. Originally designed in 1999, the site required modernization while preserving its distinctive visual character and brand identity.

Migrated from legacy HTML (font tags, table layouts, image maps) to standards-compliant XHTML/CSS architecture. Created new templates, restructured navigation, and rebuilt interactive room features from scratch. Project required approximately 40 hours to complete full redesign and content migration.

Note: This represents a transitional period in web development (2006), moving from table-based layouts to CSS-based design while responsive design was still emerging.
